# Проект To-do list

<img width="1916" height="719" alt="{53B9D817-A83C-4780-8C6B-47C6040B791E}" src="https://github.com/user-attachments/assets/7eb5a0e8-5f7d-429b-bc6c-8d79074cf03a" />

Предназначен для отслеживания задач. Реализован с помощью **Html / Css / Js**.

## Реализовано в проекте:

1. **Календарь** для отслеживания задач на этой неделе.
2. **Модуль** для добавления задач в календарь.
3. **Управление задачами:** удаление, изменение статуса и добавление задач.
4. **Подсвечивание задач разными цветами:**

| Цвет | Статус задачи |
| :--- | :--- |
| **Золотистый** | Выполнена |
| **Зелёный** | Выполняется |
| **Синий** | Частично выполнена |
| **Серый** | Не выполнена |
| **Красный** | Ближайшая задача |
| **Без выделения** | Задачи в ожидании (не подходящие под другие критерии) |

## Запуск проекта
**Локально**:
1. Перейдите в папку, в которой будет храниться проект 
```bash
cd C:\ПУТЬ
```
2. Загрузитe проект

```bash
 git clone https://github.com/Relikqq/To-do_List.git
```
3. Перейдите в папку проекта
```bash
cd C:\ПУТЬ\To-do_List
```
5. Запустите проект
```bash
start index.html
```

Запуск проекта с помощью **GitHub Pages**:
